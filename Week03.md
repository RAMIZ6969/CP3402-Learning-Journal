## Content Insights
This week, I learned how Twenty Twenty-Five is different from the older themes I have seen before. It is a block theme, which means I can use the block editor to build every single part of the site, even the header and the footer.  
I found the site editor under the appearance menu. This is where I can change the styles for the whole website at once, like picking the main colors and fonts. Using this is much faster than fixing each page one by one. It also makes sure the UniTasker site looks the same on every page, which makes it look more professional.  
For the forms, I realized that WPForms does more than just show a box for people to type in. It is actually a tool for gathering data. Every time someone fills out a form, the information is saved as a new line in the website database. This is very important for a new business. Even if the email system has a problem, the task requests from users are still safe and easy to find in the WordPress backend.

## Career/Employability/Learning Insights
This week, my view of technology changed from just using it to actually building it. I realized that a professional website is more than just a bunch of fancy pages. It is really just a way for people to interact with a database.  
Since I am studying to be a data scientist, I now see that the front part of a website is the first step of data engineering. When I set up specific boxes and rules in WPForms, I am actually cleaning the data before it even gets to the database. This makes sure the information is correct from the start.   
It showed me that a data scientist needs to understand the whole process. I need to know everything from the moment a user types their name to how that text is eventually saved in a table.
