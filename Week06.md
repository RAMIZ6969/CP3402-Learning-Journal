## Content Insights

I learned that a group project needs more than just a chat app to work well.
By committing on GitHub, we now get to know every time someone changes the code.
This helps us stay updated without needing to ask for reports. I also found that the Group Agreement is a helpful tool because it makes everyone agree on the rules and who is doing what before the work starts.
Working in a team folder on GitHub is different from working alone. I have to write clear messages when I save my work so my partners understand what I changed. Trello also helps because it shows us exactly which tasks are stuck and which ones are finished.
