## Content Insights
I learned that PHP is a server-side language, which means the code runs on the computer before the user sees anything.
This is different from HTML because PHP can make choices. For example, I learned how to use an if-statement to show a welcome message only if a user is logged in.
Another big lesson was using include files. Instead of writing the same navigation menu on every page, I can put it in one file and call it in others.
This makes the site much easier to update. I also practiced using a foreach loop to take a list of items from an array and turn them into a bulleted list automatically.  

## Career/Employability/Learning Insights
Learning PHP helps me understand how websites actually work with data. As a student interested in Data Science,
