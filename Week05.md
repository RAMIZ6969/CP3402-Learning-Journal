## Content Insights
I learned that a Child Theme is like a safety net for a website. Instead of changing the main theme files, I made a separate folder that holds all my custom changes.
This is important because when the main theme gets an update from WordPress, my changes to the colors and buttons will stay exactly as I left them.
I also found out that block themes like Twenty Twenty-Five use very specific names for things.
For example, to change a button, I had to find the exact class name in the code. I learned that CSS follows a hierarchy where the most specific rule wins. If my new code was not specific enough, the website would just keep using the original design.
